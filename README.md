# qr-code-generator
This page shows a menu for a restaurant and lists the ingredient and price of each meal. The meals are classified as Ethiopians, Chinese and Western. Although this website supports a dynamic content capablity anyone can use the static contents to get a link without payment and generate a qr code for their daily menu.
